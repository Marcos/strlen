# strlen

App to return the length of a string


## Install

```console
$ brew tap marcos/strlen
$ brew install strlen
```

## Usage

The default usage returns the length of a string
```console
$ strlen anystring
9
```

You can also show a verbose version
```console
$ strlen -v anystring
a: 1
g: 1
i: 1
n: 2
r: 1
s: 1
t: 1
y: 1
anystring: 9
```
